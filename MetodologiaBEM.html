¿Que es la Metodologia BEM

La metodologia BEM divide la interfaz de usuario en bloques independientes
para crear componentes reutilizables.
En esta metodologia no se usan ID, solo clases.
Para usarse pondremos el nombre del contenedor, por ejemplo si es un input
Pondremos class=form__input y ahi estariamos usando la metodologia BEM
Esto sirve tambien por si tenemos muchos elementos que queremos modificarlos
por igual, pero de la misma forma si queremos que el primero de todos estos
elementos sea distinto pondremos .form__input:first-child o podemos poner
.form_input--(cualquier cosa) este hace que ese input sea especial y por lo
tanto se pueda modificar sin cambiar el resto. Si tuviesemos un h2 dentro de un p la forma de seleccionarlo seria form__p-h2.
Se modifica por bloques.
